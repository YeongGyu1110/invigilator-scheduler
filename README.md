# 시험 감독 배치 프로그램

## 조건 
 - 담임인 반은 들어가지 않는다.
 - 담당 교사는 감독을 맡지 않는다.

## 상황
 - 시험 끝나고 1층으로 복귀 -> **이동 루트 최적화 불필요.**
 - 임산부, 기간제 교사, 노인에 상관없이 **모두 감독을 맡는다.**
 - 1학년, 2학년, 3학년의 시험 과목이 겹치지 않는다.
 - 컴시간 알리미에 뜨는 전체 교사 수는 **60**명이다. 다만 실무사는 뜨지 않으니 감독으로 배치 가능한 교사는 60+alpha
 - 곧 전체 교사 수, 각 교사의 이름, 교사의 담당 과목 테이블을 곧 제공받는다.

## 감독
 - 한 학년에 8개 반
 - 3학년까지 있음
 - 정감독, 부감독으로 한 반에 감독이 2명 들어옴
 - 복도에 1명이 있음. (**확실함**)

    ```
    한 층에 반(6) * 층(4) = 교내 교실 수(24)
    교내 교실 수(24) * 한 반에 들어가는 감독 수(2) = 교실에 들어가는 전체 감독 수(48)
    층당 복도 감독(1) * 층(4) = 전체 복도 감독 수(4)

    시험에 필요한 정감독/부감독 + 복도 감독 = 52명.
    ```

1. 예시로 1교시의 1학년 시험 과목이 **수학**, 2학년은 **국어**, 3학년은 **영어**일 때,\
배열 arr에 `["1수학", "2국어", "3영어"]` 처럼 과목을 넣는다.
